#  Bike Sales & Customer Analysis Dashboard

##  Project Overview
This project analyzes bike sales data to uncover trends across countries, customer age groups, and gender segments. The workflow included data cleaning and preparation in Excel, exploratory analysis using Pivot Tables and Pivot Charts, and dashboard creation in Power BI.

##  Dataset
- Bike sales dataset (Cisco lab dataset)

## Project Workflow
- Cleaned and prepared the dataset using Excel
- Performed data analysis with Pivot Tables and Pivot Charts
- Identified customer and sales trends across multiple dimensions
- Built an interactive Power BI dashboard with KPIs, charts, and slicers

##  Data Cleaning
- Identified and resolved duplicate records in **Sales_Order#**
- Missing values were handled using provided reference values from the lab dataset. No rows were deleted unnecessarily to preserve data consistency.
- Split **Product_Description** column into **Model**, **Size**, and **Color**
- Removed extra spaces and standardized **Country** values
- Replaced invalid values in **Unit_Cost** and **Unit_Price**
- Standardized Customer_Gender values (F → Female, M → Male)
- Performed spell check across text fields

##  Analysis
- Evaluated order quantity by age group segmentation
- Examined gender distribution within each customer segment
- Compared total order quantity across countries

##  Dashboard
- Total order quantity overview
- Regional distribution of orders
- Age group distribution of orders
- Gender-based segmentation of order quantity

##  Key Insights
- The United States and Australia are the top-performing markets
- Adults (35–64) are the dominant customer segment
- Female customers contribute slightly more to total sales than males
- Youth (<25) customers show the lowest engagement across most markets

This project provides a clear overview of order quantity distribution across regions, age groups, and gender segments, highlighting key customer patterns and overall demand behavior.
