#  Sales Data Analysis Project

##  Objective
Analyze sales data to identify trends and improve business decision-making.

##  Dataset
- Superstore sales dataset (from Kaggle)
- Includes orders, sales, profit, region, and dates

##  Data Cleaning
- Removed duplicate records
- Handled missing values
- Converted date column to proper format
- Standardized categorical values

##  Analysis
- Sales by region
- Profit by category
- Monthly sales trends
- Identified loss-making products

##  Dashboard
- Total Sales & Profit
- Sales by Region
- Monthly Trends
- Top Products

##  Key Insights
- West region generates the highest sales
- Technology category is the most profitable
- Some products consistently generate losses
- Sales peak during certain months

##  Tools Used
- Excel
- Power BI

##  Conclusion
The analysis highlights opportunities to improve profitability and optimize business performance.
